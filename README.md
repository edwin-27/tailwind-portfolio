# Tailwind Portfolio

## Overview

A responsive two-page portfolio website built using Tailwind CSS.

## Pages

### Home
- Hero section
- About Me section
- Education section
- Certifications section

### Gallery
- Responsive image gallery
- Grid layout using Tailwind CSS

## Technologies Used

- HTML5
- Tailwind CSS
- Git
- GitHub

## Tailwind Features Used

- Flexbox
- Grid
- Responsive breakpoints
- Utility classes
- Typography utilities
- Spacing utilities
- Hover effects
- Shadows and rounded corners

## Responsive Design

The website adapts to:
- Mobile devices
- Tablets
- Desktop screens
